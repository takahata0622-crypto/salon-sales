# B-one 売上管理アプリ 使い方ガイド

## 1. 新規登録
1. ログイン画面で「新規登録」を選ぶ
2. 「名前（本名・スペース無し）」を入力
3. 「パスワード（6文字以上）」を入力
4. 「登録して始める」をタップ

※パスワードを忘れた場合は管理者に連絡してください。

## 2. ログイン
1. ログイン画面で「ログイン」を選ぶ
2. 登録済みの名前とパスワードを入力
3. 「ログイン」をタップ

## 3. メニュー追加
1. 今日のタブを開く
2. 「メニューを追加」欄から、行ったメニュー（コース）のボタンをタップ
3. タップした分だけ下の「内訳」リストに追加される
4. 間違えたら、その項目の右の「✕」で削除できる

## 4. オプション選択
各項目の下のボタンで、伝票の内容に合わせてオン／オフを切り替える

| ボタン | 内容 |
|---|---|
| 延長 | タップごとに 0→10→20→30→0分 と切り替わる |
| 眼精疲労 | タイ・バリのメニューのみ表示（追加料金） |
| 宿泊 | 宿泊割引（10%）を適用する場合にON |
| 割引 | 日帰り割引（20%）を適用する場合にON |
| 指名料 | 指名があった場合にON（歩合100%支給） |
| 男女希望 | 男女希望があった場合にON（歩合対象外） |

設定すると、金額の右側に割引後の金額、取り消し線で割引前の金額（指名料・男女希望含む）が表示されます。

## 5. 提出（業務終了）
1. 提出タブを開く
2. 上部の部門タブで、提出したい部門（例：ボディケア）を選ぶ
3. 内容を確認し、ページ下の「業務終了（送信）」をタップ
4. 送信が完了すると「送信済み」と表示される
   - 訂正したい場合は同じ部門を選び「訂正送信」をタップすれば再送信できる

※選んでいる部門のデータだけがスプレッドシートに送信されます。他の部門も提出する場合は、部門タブを切り替えてから再度送信してください。
